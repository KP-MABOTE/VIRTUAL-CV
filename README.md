# VIRTUAL-CV
Kgaugelo Pertunia Mabote

> A responsive personal portfolio website showcasing my skills, projects, and professional background as a final-year BSc Information Technology student at North-West University.

---


## 📖 About This Project

 The site serves as a professional showcase of my academic achievements, technical skills, leadership experience, and projects.

**The goal:** To present myself as a well-rounded candidate for opportunities in Data Analytics or Business Analysis, combining technical rigour with strong communication and stakeholder skills.

---

## 🎨 Design & Visual Identity

The site pulls its visual identity from my personal brand:

| Element | Details |
| :--- | :--- |
| **Colour Palette** | Plum/wine (`#43173A`), gold (`#C9963C`), cream (`#F5EDE2`) |
| **Headings** | Georgia serif — warm, classic, confident |
| **Body Font** | Inter — clean, modern, readable |
| **Accent Font** | IBM Plex Mono — for technical/data details |
| **Motif** | Waveform-into-bar-chart — represents the two sides of my background: broadcast communication and data analysis |
